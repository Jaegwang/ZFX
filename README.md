# ZFX(Zealous FX)

ZFX는 영화 제작에 사용되는 프로그램을 설계하기 위한 라이브러리입니다.
기본적인 자료구조와 수학연산에서 부터, 복잡한 제오메트리와 선형대수까지 다양한 분야에 활용이 가능합니다.

아래 그림은 ZFX라이브러리를 활용하여 거미줄 모델링 기법을 구현한 예제입니다.
